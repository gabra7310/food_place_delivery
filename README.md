# food_place_delivery
Este projeto foi desenvolvido com base nas aulas do curso FTC - Analisando dados com Python da Comunidade DS

Este projeto visa fazer uma análise exploratória dos dados, de forma a entender o problema de negócio e desenvolver insights com base nestes dados. Este projeto também tem como objetivo criar um dashboard interativo utilizando a ferramenta Streamlit

O mesmo pode ser acessado pelo seguinte link: https://food-place-delivery2.streamlit.app/
